Tag LINK

1° There are two types of links in HTML: the A tag, which are links that take the user to other documents, and the LINK tag, which are links to external sources that will be used in the document.

1°

Existem dois tipos de links em HTML: a tag A, que são links que levam o usuário a outros documentos, e a tag LINK, que são links para fontes externas que serão usadas no documento.

2° In our example there is a LINK tag that imports the CSS for our page: The rel="stylesheet" attribute indicates that that link is relative to a file import referring to style sheets.

2°

2° Em nosso exemplo existe uma tag LINK que importa o CSS para nossa página: O atributo rel="stylesheet" indica que esse link é relativo a uma importação de arquivo referindo-se a folhas de estilo.

There are other values ​​for the REL attribute, such as ALTERNATE: In this case, we indicate to user-agents that the site's content can be found in an alternate path via Atom FEED.

3°

3° Existem outros valores para o atributo REL, como ALTERNATE: Nesse caso, indicamos aos usuários-agentes que o conteúdo do site pode ser encontrado em um caminho alternativo via Atom FEED.
