The doctype

1° The Doctype must be the first line of code in the document before the HTML tag.

2° The Doctype tells the browser and other media which code specification to use. In previous versions, it was necessary to reference the DTD directly in the Doctype code.


3° HTML5, the reference by which DTD to use is the responsibility of the Browser.


4° The Doctype is not an HTML tag, but an instruction for the browser to have information about which version of code the markup (was) written.
